## 1. 국내 해외직구·구매대행 시장 현황

### 1.1 시장 규모 및 성장 추이

| 연도 | 해외직구 거래액 | YoY 성장률 | 구매대행 비중(추정) |
|------|--------------|-----------|----------------|
| 2021 | 5조 8,000억 원 | +15% | 약 35% |
| 2022 | 6조 9,000억 원 | +19% | 약 37% |
| 2023 | 7조 8,000억 원 | +13% | 약 38% |
| 2024 | 약 8조 5,000억 원 | +9% (추정) | 약 40% |
| 2025 | 약 9조 2,000억 원 | +8% (추정) | 약 42% |

> 출처: 관세청 전자상거래 수입통계, 통계청 온라인쇼핑 동향 추정치 기반

### 1.2 주요 소비 패턴

- **소비자 동기 Top 3**:
  1. 국내 미판매 제품 (46%) — 특히 건강식품·뷰티
  2. 가격 경쟁력 (38%) — 국내가 대비 20~50% 저렴
  3. 품질·정품 선호 (16%) — 해외 직배송 정품 선호
  
- **주요 소싱 국가**:
  1. 미국 (약 55%): 건강식품, 의류, 전자기기
  2. 중국 (약 18%): 생활용품, 패션 액세서리
  3. 일본 (약 12%): 뷰티, 식품, 게임
  4. EU (약 8%): 명품, 아웃도어
  5. 기타 (약 7%)

- **연령별 이용 비율**:
  - 20대: 28% (SNS 유입, 뷰티/패션)
  - 30대: 35% (건강식품, 육아용품)
  - 40대: 25% (전자기기, 아웃도어)
  - 50대+: 12% (건강식품)

### 1.3 구매대행 시장 특이점
- 목록통관 한도 (150달러) 이내 소규모 거래 급증
- 1인 구매대행 셀러 수 증가 (스마트스토어 연계)
- 통관 강화 정책으로 식품·화장품 인증 요건 엄격화 추세
- 고환율 지속으로 원가 부담 증가 → 마진 관리 중요성 부각

## 2. 경쟁사 분석

### 2.1 직접 경쟁사

#### iHerb (iherb.com)
| 항목 | 내용 |
|------|------|
| 유형 | 해외 직판 플랫폼 (구매대행 아님) |
| 강점 | 방대한 상품 수(3만+), 저렴한 가격, 한국어 지원 |
| 약점 | 직구 경험 없는 소비자 진입장벽, CS 한계, 환불 번거로움 |
| 월 방문자 | 전 세계 약 2,000만 명, 한국 추정 100만+ |
| Sniper 대응 | iHerb보다 편의성 강조, 국내CS·통관 대행 차별화 |

#### 몰테일 (malltail.com)
| 항목 | 내용 |
|------|------|
| 유형 | 배송대행지(배대지) 서비스 |
| 강점 | 낮은 배송비, 브랜드 인지도, 합배송 기능 |
| 약점 | 소싱 서비스 없음 (배송만), 상품 큐레이션 없음 |
| 타겟 | 이미 해외직구 경험 있는 사용자 |
| Sniper 대응 | 몰테일은 경쟁사가 아닌 보완재 — 소싱부터 대행하는 점이 차별점 |

#### 네이버 쇼핑 글로벌 (구매대행 셀러들)
| 항목 | 내용 |
|------|------|
| 유형 | 스마트스토어 입점 개인 구매대행 셀러 |
| 강점 | 네이버 트래픽, 네이버페이 신뢰 |
| 약점 | 상품 정보 불투명, CS 품질 편차 크음, 가격 덤핑 경쟁 심화 |
| 가격 | 마진 박탈 수준 가격 경쟁 다수 |
| Sniper 대응 | 신뢰도·상세정보·CS 품질 차별화. 가격 경쟁 대신 큐레이션 경쟁 |

#### 무신사 글로벌 (musinsa.com)
| 항목 | 내용 |
|------|------|
| 유형 | 패션 전문 해외 브랜드 공식 수입/판매 |
| 강점 | 강력한 20-30대 패션 트래픽, 브랜드 파워 |
| 약점 | 패션 카테고리에 집중, 건강식품/운동용품 약함 |
| Sniper 대응 | 카테고리 겹침 적음. 건강식품·운동·아웃도어에 집중 |

### 2.2 간접 경쟁사

| 플랫폼 | 포지셔닝 | Sniper 대비 약점 |
|--------|---------|----------------|
| 쿠팡 로켓직구 | 빠른 배송, 낮은 가격 | 큐레이션 없음, 마진율 투명성 없음 |
| 11번가 아마존 직구 | 아마존 연동 | 영어 상품명, 소비자 직접 탐색 필요 |
| 위메프/티몬 구매대행 | 플래시 딜 | 플랫폼 불안정, 신뢰도 낮음 |
| 개인 인스타마켓 | SNS 기반 판매 | 공식 결제·환불 체계 없음 |

### 2.3 포지셔닝 맵

```
           [고가/프리미엄]
                ▲
                │ 무신사글로벌
                │     ●
   [낮은 큐레이션]─────────────[높은 큐레이션]
                │
   스마트스토어  │      ● SNIPER (목표)
   개인셀러●    │
                │
           ● iHerb   ● 몰테일
                ▼
           [저가/가성비]
```

**Sniper의 목표 포지션**: 높은 큐레이션(Sniper Score) + 중간 가격대(합리적 구매대행 수수료)

### 3.2 카테고리 상세 분석

#### 카테고리 1: 건강식품 (비타민/프로틴/오메가3)

**시장 현황**
- 국내 건강기능식품 시장: 약 6조 원 (2024년 추정)
- iHerb 한국 직접 구매: 연 수십만 건 이상 추정
- 국내 판매가 대비 iHerb 가격: 일반적으로 30~60% 저렴

**핵심 인기 상품 예시**
| 상품명 | iHerb 가격 | 국내 유사 제품가 | 잠재 마진율 |
|--------|----------|--------------|-----------|
| Now Foods Vitamin D3 5000IU | $8-12 | 25,000-35,000원 | 35-45% |
| Optimum Nutrition Gold Standard Whey 2.27kg | $45-55 | 100,000-120,000원 | 25-35% |
| Nordic Naturals Omega-3 | $18-25 | 50,000-70,000원 | 30-40% |

**통관 주의사항**
- 건강식품은 식약처 건강기능식품 인증 필요 (상업용 수입 시)
- 개인 사용 목적 목록통관 가능 (150달러 이하, 6병 이하)
- 구매대행 시 개인 용도로 신고 → 소량 반복 주문 전략

**Sniper Score 예상**
- 국내 수요: 19/20점, 해외 가격 경쟁력: 18/20점, 통관리스크: 6/10점
- 예상 평균 Sniper Score: **78-82점** → 등록 강력 권장

## 5. SWOT 분석

### 강점 (Strengths)
| 항목 | 내용 |
|------|------|
| Sniper Score 독자 지표 | 경쟁사 없는 수익성 검증 시스템 |

| 타겟 카테고리 명확 | 건강식품/아웃도어에 집중, 전문성 확보 용이 |

### 약점 (Weaknesses)

## 6. 시장 진입 전략

### 6.1 초기 집중 카테고리 (MVP)
```
1순위: 아웃도어/캠핑 (Sniper Score 85점, 통관 리스크 낮음)
2순위: 건강식품 비타민 (Sniper Score 80점, 수요 최대)
3순위: 운동용품 (Sniper Score 76점, 경쟁 중간)
```

### 6.2 차별화 포인트
1. **투명한 마진 공시**: "우리는 이렇게 마진을 법니다" — 신뢰 구축
2. **Sniper Score 공개**: 왜 이 상품을 파는지 근거 제시
3. **통관 보증**: 통관 불가 시 100% 환불 정책
4. **전문 CS**: 제품별 상세 사용 안내 + 질문 응대

### 6.3 성장 경로 (Flywheel)
```
좋은 상품(Sniper Score) 
    → 높은 마진율 
        → 광고 재투자 가능 
            → 더 많은 트래픽 
                → 더 많은 주문 
                    → 데이터 축적 
                        → 더 정확한 Sniper Score
```
