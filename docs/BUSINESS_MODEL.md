## 1. 비즈니스 모델 캔버스

### 🏆 가치 제안 (Value Proposition)

#### 소비자를 위한 가치
- **접근성**: 영어·일본어 못해도 해외 유명 브랜드 제품 구매 가능
- **신뢰성**: Sniper Score 70점 이상 검증 상품만 판매 → 사기·저품질 위험 최소화
- **편의성**: 통관, 관세, 국제배송 모든 복잡한 과정을 대신 처리
- **가격 경쟁력**: 국내 미발매 또는 국내가 대비 20% 이상 저렴한 해외 상품 큐레이션
- **안전 보장**: 파손/분실 시 환불 정책, 통관 불가 시 100% 환불

#### 운영자(자신)를 위한 가치
- **수익 투명성**: 마진 계산기로 매 거래 수익을 정확히 파악
- **운영 효율**: 1인이 관리 가능한 자동화 중심 구조
- **리스크 관리**: Sniper Score로 팔리지 않는 상품 사전 차단

### 💰 수익 모델 (Revenue Streams)

#### Phase 1 — 구매대행 직접 마진
```
수익 구조:
  국내 판매가 (소비자 지불)
  - 총원가 (해외원가 + 배송 + 관세 + 수수료)
  = 순마진 (운영자 수익)

목표: 마진율 최소 30% 유지
```

| 수익원 | 설명 | 예시 (마진율 30%기준) |
|--------|------|-------------------|
| 상품 마진 | (판매가 - 원가) | 3만원 판매 → 9,000원 마진 |
| 배송비 마진 | 실제 배송비보다 약간 높게 청구 | 실비 3,000원 → 청구 4,000원 |
| 빠른배송 옵션 | 항공 특급 배송 추가요금 | +5,000~10,000원 |

- 월 20건 × 평균 마진 15,000원 = 300,000원
- 월 50건 × 평균 마진 15,000원 = 750,000원

## 4. 단위 경제학 (Unit Economics)

### 예시 상품: 미국산 멀티비타민 (iHerb 소싱)

```
해외원가:          $12.00 = 약 16,800원
현지배송비:         $0 (iHerb 기본 무료)
국제배송비:        약 5,000원
관세:              0원 (식품, 목록통관 면세)
부가세:            0원 (목록통관 면세)
국내배송비:        3,000원
결제수수료(3%):    약 960원
기타비용:          500원
─────────────────────────────
총원가:            26,260원

국내 판매가:       32,000원

예상 순마진:       32,000 - 26,260 = 5,740원
마진율:            5,740 / 32,000 × 100 = 17.9%  ← 목표치 미달

→ 조정: 판매가를 38,000원으로 올리거나, 더 비싼 상품 선택 권장
→ Sniper Score 마진율 항목 0점 → 재검토 필요
```

### 마진율별 상품 분류

| 마진율 | 판단 | Sniper Score 마진 점수 |
|--------|------|----------------------|
| < 20% | 판매 금지 | 0점 |
| 20~25% | 재검토 권장 | 5점 |
| 25~30% | 최소 허용 | 8점 |
| 30~40% | 권장 | 12점 |
| 40%+ | 최우선 등록 | 15점 |
