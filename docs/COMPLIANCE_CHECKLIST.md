> ⚠️ 이 문서는 참고용입니다. 최종 법적 판단은 변호사·세관에 문의하세요.

## 1. 사업자 등록 및 통신판매업 신고

### 1.1 사업자 등록

- [ ] 개인사업자 또는 법인사업자 등록 완료 (관할 세무서)
- [ ] 업태: 소매업 / 업종: 전자상거래 소매업 (또는 해외 구매대행업)
- [ ] 사업장 주소 확정 (자택 가능)
- [ ] 사업자등록증 발급 완료

### 1.2 통신판매업 신고

- [ ] 통신판매업 신고서 제출 완료 (관할 시·군·구청)
- [ ] 공정거래위원회 통신판매업 신고번호 취득
- [ ] 쇼핑몰 하단에 통신판매업 신고번호 표시 완료
  - 필수 표시: 상호명, 대표자명, 사업자등록번호, 통신판매업 신고번호, 주소, 전화번호, 이메일
- [ ] 에스크로 또는 소비자피해보상보험 가입 완료 (연 매출 1억 미만은 면제 가능)

### 1.3 구매대행 사업 특이사항

**구매대행 vs 직수입 법적 차이**

| 구분 | 구매대행 | 직수입 |
|------|---------|------|
| 수입자 | **고객(개인)** | 사업자 |
| 관세 납부 의무자 | 고객 | 사업자 |
| 개인통관고유번호 | **필수** | 불필요 |
| 수입신고 | 고객 명의 | 사업자 명의 |
| 적용 한도 | 목록통관 $150 | 일반수입 기준 |
| 부가세 | 고객 부담 | 사업자 납부 |
| 소비자보호법 적용 | 제한적 | 전면 적용 |

- [ ] 구매대행 방식으로 사업 구조 확정 (고객이 수입자)
- [ ] 계약서/주문서에 "구매대행" 명시 의무 확인 완료
- [ ] 서비스 수수료(구매대행 수수료) 별도 항목으로 표시 여부 검토

## 2. 개인통관고유번호 수집·관리

### 2.1 수집 의무

- [ ] 모든 주문 시 개인통관고유번호 수집 프로세스 구현 완료
- [ ] 개인통관고유번호 미제출 시 주문 불가 처리 로직 설정
- [ ] 개인통관고유번호 형식 검증: P + 영문대문자 + 8자리 숫자 (예: PA12345678)
- [ ] 관세청 개인통관고유번호 조회 API 연동 또는 수동 검증 절차 마련

### 2.2 개인통관고유번호 발급 안내

- [ ] 쇼핑몰에 개인통관고유번호 발급 안내 페이지 또는 링크 제공
  - 발급처: 관세청 전자통관시스템 (unipass.customs.go.kr)
  - 발급 시간: 즉시 발급
  - 발급 비용: 무료
- [ ] 고객 FAQ에 개인통관고유번호 관련 질문 추가 완료

### 2.3 데이터 보관

- [ ] 개인통관고유번호 Supabase 암호화 저장 (AES-256 또는 pgcrypto)
- [ ] 해관 제출 후 불필요 시 삭제 정책 수립
- [ ] 개인통관고유번호 제3자 제공 금지 (세관 제출 목적 외)

## 3. 관세 및 부가세 기준

### 3.1 목록통관 기준 (구매대행 주요 적용)

- [ ] 목록통관 한도 $150 (USD) 이하 상품 확인
  - 합산과세: 동일 수입자(고객)가 같은 날 여러 건 수입 시 합산
  - 물품 가격 + 현지 운송비 기준 (국제 배송비 제외 가능)
- [ ] $150 초과 시 일반수입신고 안내 자동화 여부 확인
- [ ] 합산과세 위험 고지 문구 주문 페이지 표시 여부

### 3.2 세율 기준 (카테고리별)

| 카테고리 | 관세율 | 부가세 | 개별소비세 | 비고 |
|---------|------|------|--------|------|
| 건강기능식품 | 8% | 10% | - | 식품안전처 허가 필요 |
| 화장품 | 6.5% | 10% | - | 기능성 여부 확인 |
| 의류/패션 | 13% | 10% | - | |
| 전자기기 | 0~8% | 10% | - | KC 인증 필요 |
| 향수/기타 | 6.5% | 10% | 7% | 개소세 적용 |
| 식품 (일반) | 8~27% | 10% | - | 검역 필요 가능 |

- [ ] 상품별 HS 코드 확인 및 실제 관세율 검증 완료
- [ ] 주문 페이지에 "관부가세는 고객 부담" 명시 완료
- [ ] DB의 `tax_estimate` 필드에 예상 세금 자동 계산 로직 구현

### 3.3 면세 기준

- [ ] $150 이하 + 목록통관 요건 충족 시 면세 적용 안내
- [ ] 자가사용 목적 소명 관련 고객 안내 문구 준비

### 4.4 전자기기

- [ ] KC 인증 필요 품목 목록 확인 (전기용품 및 생활용품 안전관리법)
  - 필수: 충전기, 배터리, 전동 기기, 조명 기기 등
- [ ] KC 인증 없는 전자제품: 개인 자가용 1개 한정 통관 가능, 판매 금지
- [ ] KC 미인증 제품 판매 시 전파법·제품안전기본법 위반 주의

| 카테고리 | 리스크 등급 | 주요 위험 요소 | 취급 권장 여부 |
