## 2. 마진 계산 정확도 검증

- [ ] 총원가 공식 검증: `해외원가(KRW) + 현지배송비(KRW) + 국제배송비 + 관세+부가세 + 국내배송비 + 결제수수료 + 기타비용`
- [ ] 해외원가 KRW 환산: `overseasPrice × exchangeRate` 소수점 반올림 확인
- [ ] 현지배송비 KRW 환산: `localShippingCost × exchangeRate` 소수점 반올림 확인
- [ ] 예상순마진: `국내판매가 - 총원가` 음수 가능 여부 확인
- [ ] 마진율: `(예상순마진 / 국내판매가) × 100` 소수점 2자리 확인
- [ ] 국내판매가 0 입력 시 마진율 0으로 처리되는지 확인 (ZeroDivisionError 방지)
- [ ] 나우푸즈 비타민D3 샘플: 총원가 24,575원, 마진율 29.8% 검증
- [ ] 앤커 충전기 샘플: 음수 마진율 (-7.0%) 표시 확인

## 3. Sniper Score 로직 검증

- [ ] 국내수요 1점 → 4점, 5점 → 20점 매핑 확인
- [ ] 마진율 30% 이상 → 20점, 10% 미만 → 0점 확인
- [ ] 통관리스크 LOW → 10점, MEDIUM → 6점, HIGH → 2점 확인
- [ ] 경쟁강도 낮음 → 5점, 보통 → 3점, 높음 → 1점 확인
- [ ] 배송안정성 1점 → 3점, 5점 → 15점 매핑 확인
- [ ] 총점 합계 검증: breakdown 8개 항목 합 = total 일치 확인
- [ ] 스나이퍼 스코어 상위 상품 정렬 순서 확인 (세라비 82점 1위)
- [ ] 마진 계산기에서 마진율 변경 시 스나이퍼 스코어 실시간 업데이트 확인
